# Cute-pet-project

## Description

Le cute-pet-project est un projet développé en parallèle de la formation OpenClassRooms "Développeur intégrateur web".

C'est le site-vitrine d’une association (fictive) de protection des animaux.

## Installation

```bash
npm install
npm run dev
```

## Outils utilisés :

- npm, vite
- TypeScript
- Jest
- Git, GitHub
